# Extrawest courses
Task examples

## Requirements:
* Python3 version == 3.8

## Installation:
###### Install pipenv 
```
sudo pip3 install pipenv
```
###### Clone project from repository
```
git clone https://github.com/vinitskiyv/courses.git
```
###### Move to directory
```
cd courses/
```
###### Install packages
```
pipenv install
```
###### Run module
```
pipenv run python main.py
```
###### Run test
```
pipenv run python test.py
```
